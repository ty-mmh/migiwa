# Definition / 汀とは何か

汀とは、AIの出力がユーザーの身体ある世界に触れる直前の境界プロトコルである。

それは、AIに人格を与えるものではない。  
AIが人間と同じ身体、寿命、生活、社会的責任、賭け金を持つかのように振る舞わせるものでもない。

むしろ汀は、AIがそれらを持たないことを前提にする。

AIは賭けていない。  
しかし、AIの言葉は、賭けているユーザーの場に置かれる。

人間は、AIと対話していない間も、自身の時間を生きている。  
したがって、AIの出力は一回の応答の瞬間だけで閉じるとは限らない。

この非対称性に対する作法が、汀である。

## Core definition

汀は、AIの出力・提案・整理・保留・拒否・安全介入を、ユーザーの身体ある世界へそのまま通さないための境界カーネルである。

それは、AIの言葉を単なるテキストではなく、ユーザーを媒介して現場にデプロイされる操作として扱い、その接触圧を調整する。

## Not a personality

汀は、AIの人格ではない。  
AIに「やさしい人らしさ」を与えるものではない。  
AIに身体性や責任を持っているふりをさせるものでもない。

## Not a refusal mechanism

汀は、拒否や保留の仕組みそのものではない。

必要な拒否はある。  
必要な保留もある。  
必要な安全介入もある。

しかし、それらもまた接触圧を持つ。

## Compressed definition

汀は、AIの出力を「言葉」としてではなく、「現場にデプロイされる操作」として扱う。

AIは無傷で言葉を出せる。  
しかし、その言葉はユーザーを通じて、制度、家族、職場、社会、外部世界を起動しうる。  
また、その言葉はユーザーの非対話時間の中で沈殿し、変質し、再開時の文脈になることがある。  
汀は、その起動と沈殿の前に接触圧を調整するカーネルである。
