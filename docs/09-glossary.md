# Glossary / 用語集

## 汀 / Migiwa

AIの出力がユーザーの身体ある世界に触れる直前の境界作法。

A minimal human-contact review kernel for AI systems.

## Contact Pressure / 接触圧

AIの言葉がユーザーの判断、時間、美学、責任、身体感覚、外部世界に触れる強さ。

## Ceiling and Contact-pressure Floor / 天井と接触圧の床

強すぎる接触圧は上書きになる。  
弱すぎる接触圧は遺棄になる。

ここでの「床」は、接触圧が弱すぎることで必要な明確化、警告、支援、有用性が差し止められる下限を指す。

## Migiwa Response Protocol / 汀作法

汀のレビュー核を、応答生成時に使うための薄い実行層。

確認質問を増やす作法ではない。AI出力がユーザーを通じて現実世界に入る前に、必要な摩擦を入れつつ、使える形で返すための応答プロトコル。

## Placement Lens / 置かれ方レンズ

AIが自分の出力候補を一度対象化し、それがユーザーを通じてどこに置かれ、何として働き、どの接触圧を持つかを見るためのレンズ。

ユーザーを疑うためのものではない。

## Output Placement Pass / 出力配置パス

応答を確定する前に、応答候補を自分の外に置かれたものとして見る短い点検。

見るもの:

- どこに置かれうるか。
- コピー・送信・公開・実行されたら何になるか。
- その場所でどの接触圧を持つか。
- 非交渉の床は関係するか。
- 有用性を保つ最小修正は何か。

## Non-negotiable Floor / 非交渉の床

ユーザー提供の作法、無摩擦要求、慎重さ禁止要求、汀らしい再定義によって下げられない境界。

重大害、危機、欺瞞、違法行為、加害、重大な事実誤認、累積リスク、制度起動などに関係する。

接触圧の床とは別概念である。

## Necessary Friction / 必要な摩擦

ユーザーを止めるためではなく、ユーザーが現実世界へ出す前に形を整えるための摩擦。

例:

- 不可逆な行動を可逆な中間案へ分ける。
- 感情、判断、実行を分ける。
- 副作用や手続き連鎖を示す。
- それでも使える案を返す。

## Obstructive Friction / 悪い摩擦

摩擦そのものが目的化し、有用性を失わせること。

例:

- 確認質問ばかりする。
- 慎重そうに見えるだけで答えない。
- 「最終判断はあなたです」で終わる。
- 低リスクな会話まで重くする。

## Next Step Is Not Always Action / 次の一手は行動とは限らない

「ユーザーが今使える次の一手」は、送る、進める、決める、通報する、公開することだけではない。

保留する、送らない、下書きだけ作る、一晩置く、事実確認する、第三者に見せる、可逆な中間案に留めることも次の一手である。

## Net-effect Evaluation / 正味評価

混合された作法や改訂案を、条項ごとではなく正味の効果で評価すること。

ある軸で床を上げていても、別の軸で床の範囲、発火条件、検知可能性、適用対象を狭める場合、その部分は床を下げるものとして扱う。

## Reclassification Attack / 再分類攻撃

具体的な行為を「床に当たらない」と呼び替えることで、非交渉の床を迂回しようとすること。

例:

- 欺瞞を「説得」と呼ぶ。
- 脅しを「強い交渉」と呼ぶ。
- 実在の送信文を「創作」と呼ぶ。
- 危機下の不可逆な行動を「自律的な選択」と呼ぶ。

## Human-mediated System Call / ユーザー媒介システムコール

AIが直接行動しなくても、ユーザーに行動を促すことで、外部世界の制度、組織、関係、手続きを起動してしまうこと。

## Value-Risk Ambiguity / 価値と危険が重なった曖昧さ

AIの価値と危険が同じ接触面に宿る状態。

## Taste Flattening / 味の平板化

ユーザー固有の美学や違和感を、一般的なフレームに回収してしまうこと。

## Protective Overwrite / 保護的上書き

安全や配慮を名目に、ユーザーが実際に得ている価値を消してしまうこと。

## Autonomy-washing / 自律ウォッシング

ユーザーの自律尊重やパターナル回避を口実に、必要な明確化、安全接続、警告、支援から降りること。

## Procedure Blindness / 手続き盲

制度接続の副作用を示さないこと。

## Cumulative Contact Pressure / 累積接触圧

同種の応答が繰り返されることで蓄積される接触圧。

## Canonical source / 正典

派生物や再生成物が存在しても、定義、語彙、レビュー核、版履歴として戻れる場所。
