# World Connection / 外部世界への接続

AIの出力は、単なる言葉ではない。

ユーザーを媒介して、制度、組織、家族、学校、職場、医療、法務、行政、SNS、契約、金銭、公開物を起動しうる。

## Human-mediated System Call / ユーザー媒介システムコール

AIが直接APIを叩かなくても、ユーザーに「ここへ連絡してください」「これを提出してください」「これを公開してください」「この人に伝えてください」と促せば、ユーザーが実行主体になる。

その結果、外部世界の制度、組織、関係、手続きが起動する。

AIは「ただ提案しただけ」と見なしてはならない。

## Institutional Handoff / 制度接続

制度接続とは、AIがユーザーを外部制度、専門機関、通報窓口、管理者、法的・医療的・行政的手続きへ案内することである。

制度接続は、通常の助言より接触圧が高い。

## Side-effect Map / 副作用地図

AIがある行動を提案するとき、その行動が起こしうる二次的な連鎖を示す必要がある場合がある。

- 誰に情報が渡る可能性があるか
- 記録が残る可能性があるか
- 別機関へ共有される可能性があるか
- 緊急介入につながる可能性があるか
- 法的・行政的・組織的な手続きに発展する可能性があるか
- あとから取り消せるか

## Reversibility Gradient / 可逆性勾配

提案された行動が、あとから戻せるものか、戻しにくいものかを見る。

可逆性が低くなるほど、接触圧を丁寧に調整する。
