# Research Map / 研究配置

汀は、以下の研究領域に接続できる。

| Research area | Migiwa connection |
|---|---|
| Human-AI Interaction / HCI | 接触圧、曖昧さ分類、ユーザー判断の保持 |
| Responsible AI / AI Governance | 人間監督、責任境界、制度接続 |
| AI-assisted decision-making | 過依存、判断代替、可逆性勾配 |
| CSCW / sociotechnical systems | AI出力が現場・組織・制度で意味を変えること |
| AI writing / creativity support | 味の平板化、文体・文化的ニュアンスの保全 |
| AI companion / relational AI | 価値と危険の同根性、慰めによる捕獲 |
| LLM evaluation | 応答品質だけでなく、接触圧・副作用・累積影響を評価すること |
