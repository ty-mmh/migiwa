## v0.2.0 — Migiwa-type Intelligence Augmentation

Adds the first major derived protocol built from the Migiwa contact-pressure kernel.

Includes:

- Migiwa-type Intelligence Augmentation / 汀型・知性拡張
- Thought Externalization Playbook / 思考外部化プレイブック
- Japanese and English thought-externalization protocol prompts
- Response Protocol / 汀作法 runtime layer
- Non-negotiable Floor / 非交渉の床
- Placement Lens / 置かれ方レンズ
- Runtime Examples / 応答時の例
- Additional runtime failure modes, including placement blindness, floor-lowering by protocol, and action-biased next step
- Clarification that Migiwa-type Intelligence Augmentation is a derived protocol, not a replacement for the minimal kernel
- The operating formula: build the scaffold; do not own the landing

Initial public placement of Migiwa as a minimal human-contact review kernel for AI systems.

Includes:

- Minimal kernel
- Contact pressure with ceiling and floor
- Failure modes, including autonomy-washing and protective overwrite
- World connection and human-mediated system call
- Japanese and English review prompts
- Dual licensing by file type
- Canonical source and AI-use notice
