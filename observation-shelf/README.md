# Observation Shelf / 観測棚

This directory stores dated observations around Migiwa and its neighboring AI contexts.

このディレクトリは、汀とその周辺にあるAI文脈についての日付入り観測を置く棚である。

These notes are not canonical kernel documents.
They are situated observations: snapshots of how Migiwa appears when placed against the world at a specific date.

これらの文書は、汀の正典カーネルそのものではない。
特定の日付において、汀を世界のAI文脈に置いたときに見える差分、摩擦、近隣、違和感を記録する観測である。

## Role

The canonical documents define Migiwa.
The observation shelf records how Migiwa is positioned against changing external contexts.

正典文書は、汀を定義する。
観測棚は、変化する外部文脈に対して、汀がどこに位置して見えるかを記録する。

## Status

Observation shelf documents may later inform:

- canonical docs
- research notes
- playbooks
- release notes
- external essays

But they should not be treated as canonical merely because they are placed in this repository.

観測棚の文書は、後に正典文書、調査ノート、プレイブック、リリースノート、外部向けエッセイの素材になりうる。
しかし、このリポジトリに置かれていること自体によって、正典として扱われるべきではない。

## Naming

Each observation should include a date.

日付を含める。

Recommended format:

```text
YYYY-MM-DD-topic.md
```

Example:

```text
2026-07-02-ai-world-diff.md
```
