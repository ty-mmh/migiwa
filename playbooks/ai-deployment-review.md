# Playbook: AI Deployment Review

Use this playbook when AI outputs enter real workflows, organizations, tools, customer experiences, or production processes.

## Review questions

1. What user action may this AI output cause?
2. Is the action reversible?
3. Does the output imply a decision rather than a proposal?
4. Does it trigger a human-mediated system call?
5. Does it map side effects?
6. Is the response too forceful or too weak?
7. What happens if this pattern repeats 100 times?

## Deliverable

A Contact Pressure Review should include:

- contact surfaces
- ceiling risks
- floor risks
- failure modes
- reversibility gradient
- side-effect map
- suggested response revisions
